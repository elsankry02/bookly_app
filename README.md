# 🚀 Features
* **Explore Books:** Browse various book categories and the latest releases.
* **Search Functionality:** Real-time search to find specific books using Google Books API.
* **Book Details:** View detailed information about each book, including descriptions and ratings.
* **Preview Books:** Directly open and preview books within the app.
* **Responsive UI:** A clean and modern design that adapts to different screen sizes.

# 🛠️ Tech Stack & Tools
* **State Management:** Flutter Bloc / Cubit.
* **Networking:** Dio (for fetching data from Google Books API).
* **Navigation:** GoRouter (for declarative and efficient routing).
* **Architecture:** MVVM (Feature-based folder structure).
* **Service Locator:** GetIt (for dependency injection).
* **Data Parsing:** Manual JSON serialization into Dart models.
